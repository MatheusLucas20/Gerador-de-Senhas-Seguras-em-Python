# 🔐 Gerador de Senhas Seguras em Python

Um gerador de senhas simples, robusto e criptograficamente seguro desenvolvido em Python. O script roda direto no terminal e permite que o usuário escolha o tamanho da senha desejada, contando com validação de erros para evitar falhas de digitação.

---

## ✨ Funcionalidades

*   **Segurança Forte:** Utiliza o módulo `secrets` do Python, garantindo que as senhas sejam geradas de forma criptograficamente segura (ideal para senhas e tokens).
*   **Variedade de Caracteres:** Combina letras maiúsculas, minúsculas, números e caracteres especiais/símbolos.
*   **Validação de Entrada:** Proteção contra digitação de letras, caracteres vazios, números negativos ou zero no campo de tamanho da senha.

---

## 🛠️ Tecnologias Utilizadas

*   **Python 3.x**
*   Módulos Nativos:
    *   `secrets` (Geração aleatória segura)
    *   `string` (Manipulação de cadeias de caracteres)
